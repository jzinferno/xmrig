# XMRig

XMRig is a high performance, open source, cross platform RandomX, KawPow, CryptoNight and [GhostRider](https://github.com/jzinferno/xmrig/tree/master/src/crypto/ghostrider#readme) unified CPU/GPU miner and [RandomX benchmark](https://xmrig.com/benchmark). Official binaries are available for Windows, Linux, and Android.

## Mining backends
- **CPU** (x86/x86_64/ARMv7/ARMv8)
- **GPU** (OpenCL for AMD GPUs only)

## Usage
```bash
./xmrig --configure
./xmrig --clean
./xmrig --show
./xmrig
```
For a detailed description of all available parameters and usage examples, please refer to the [official XMRig documentation](https://xmrig.com/docs/miner/command-line-options)
